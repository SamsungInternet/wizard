---
title: Samsung Internet vous protège
---

Mesures prises par Samsung Internet pour assurer votre sécurité par défaut, vous pouvez trouver la plupart de ces paramètres dans :

Paramètres ➔ Confidentialité et sécurité

Ceux-ci devraient tous être activés pour une expérience de navigation des plus sûres :

### Bloquer les redirections vers l'arrière

Les sites Web malveillants essaient parfois de vous empêcher de quitter ou de vous envoyer vers d'autres pages lorsque vous essayez de quitter en appuyant sur le bouton retour. Cela fonctionne en insérant des entrées malveillantes dans votre pile d'historique. 

Ce paramètre empêche cette attaque en bloquant cette capacité.

### Bloquer les fenêtres pop-up

Cette fonction bloque les pages Web qui tentent d'ouvrir de nouvelles fenêtres et de nouveaux onglets en arrière-plan lorsque vous naviguez sur le Web.

### Bloquer l'ouverture d'Internet par des applications

Certaines applications malveillantes sur votre téléphone tenteront d'attirer votre attention en ouvrant à plusieurs reprises votre navigateur Web par défaut sur une page Web de spam. Si vous réglez votre navigateur Web par défaut sur Samsung Internet, vous pouvez alors empêcher ces applications d'ouvrir le navigateur afin qu'elles ne puissent pas vous spammer.

En appuyant sur ce paramètre, vous obtiendrez une liste de toutes les applications qui ont récemment ouvert le navigateur Web, le nombre de fois que chaque application a ouvert le navigateur, puis vous pourrez bloquer l'application.

### Smart Anti-Tracking (Anti-traçage intelligent)

Assurez-vous qu'il est réglé sur "Toujours".

L'anti-traçage intelligent est abordé plus en détail dans la section [Smart Anti Tracking](#smart-anti-tracking).

### Avertir des sites malveillants

Cette option doit être activée.

Voir [Malicious Websites](#malicious-websites) pour plus d'informations.

### Bloquer les téléchargements automatiques

Empêche les sites Web d'essayer de télécharger sournoisement des fichiers dangereux.

### Protection contre l'empreinte digitale

La protection par empreinte digitale n'est pas une fonction à activer ou à désactiver. Il s'agit d'un principe de conception dans la manière dont le navigateur est construit.

On parle d'empreinte digitale lorsque les sites Web collectent des données en fonction de la façon dont vous interagissez avec votre navigateur ou des propriétés uniques de votre téléphone. Si un site web peut identifier de manière unique cet appareil, il peut suivre votre retour ou même votre navigation sur d'autres sites dans certains cas.

Nous essayons d'empêcher les sites web d'identifier votre appareil de manière unique en fonction des informations relatives à votre navigateur/appareil. Il s'agit d'un processus continu qui s'améliore avec chaque nouvelle version du navigateur.
